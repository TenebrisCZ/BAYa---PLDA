# BAYa - Probabilistic Linear Discriminant Analysis (PLDA) Inference

## Overview

This repository contains my solution and materials for the **BAYa Course** assignment at Brno University of Technology, focusing on the Probabilistic Linear Discriminant Analysis (PLDA) model. The primary objective of this project is the implementation and analysis of inference for the PLDA model, as described in the course lectures.

The core of this work is implemented within a Jupyter Notebook, which includes data generation, visualization, and the completed code for the PLDA model inference.

## Prerequisites & Setup

To work on this assignment, you will need an environment capable of running Jupyter Notebooks.

## Notebook Structure

The `plda.ipynb` notebook is organized into several sequential sections:

* **Helper Functions:** The first cell contains functions for presenting results and visualizing the learned models. You can initially skip reading this cell, as the usage of these functions will be demonstrated later in the notebook.
* **Data Generation:** The next section contains code that "handcrafts" specific parameters of the PLDA model and implements its assumed generative process. This code generates the artificial training data you will use to train your PLDA model. **Please read this code and its accompanying comments carefully.**
* **Implementation Tasks:** Throughout the notebook, you will find designated cells with instructions. This is where you will write your code to implement the missing components of the PLDA model.
* **Analysis & Questions:** Along with coding tasks, there are text fields containing specific questions to answer and analytical tasks to accomplish based on your implementation and results.

## Instructions

1. Read the background material on PLDA provided in the BAYa class slides.
2. Open the notebook and run the initial setup and data generation cells.
3. Follow the instructions within the notebook to fill in the missing implementation details.
4. Complete all analysis tasks and answer the questions directly within the provided fields in the notebook.
